# 個人履歷網站

這是一個使用 Next.js（React）、TypeScript 與 Tailwind CSS 製作的個人履歷/作品集網站。

## 功能特色
- 首頁、部落格、文章、專案等多頁面
- 響應式設計，適合各種裝置
- 元件化架構，方便擴充與維護

## 安裝與啟動方式

1. **安裝依賴套件**

   請先安裝 [pnpm](https://pnpm.io/)。
   
   在終端機輸入：
   ```bash
   pnpm install
   ```

2. **啟動開發伺服器**

   ```bash
   pnpm dev
   ```

3. **瀏覽網站**

   開啟瀏覽器並前往：
   [http://localhost:3000](http://localhost:3000)
